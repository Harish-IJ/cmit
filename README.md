# cmit

Dev quickstart:

1. Install deps: npm install
2. Start dev mode: npm run dev
3. Or build: npm run build && npm start

Usage:
npx cmit # interactive flow
npx cmit -q feat "message" # quick mode
npx cmit --push # commit then push
npx cmit --amend # amend last commit

Config:
Place `.cmitrc.json` in project root to override commit types and toggles.

Stats:
A local `.cmit-stats.json` file is created to track commit type usage.
